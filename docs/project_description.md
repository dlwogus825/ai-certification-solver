# AI 활용 자격증 학습 지원 시스템 개발

## 프로젝트 개요
본 프로젝트는 인공지능(AI) 기술을 활용하여 자격증 준비생을 위한 맞춤형 문제 풀이 웹 플랫폼을 개발하는 것을 목표로 합니다. 사용자 업로드 기출문제 분석, 유사 문제 자동 생성, LLM(Large Language Model) 기반 문제 해설 및 학습 가이드 제공 등을 통해 개인별 취약점 분석 및 효율적인 학습 방향을 제시합니다.

## 주요 기능
*   **사용자별 자격증 선택 및 개인화 학습 대시보드 제공**
*   **AI 기반 기출문제 분석 및 유사 문제 자동 생성 기능**
*   **자격증별 문제은행 및 실시간 문제 풀이 시스템**
*   **개인별 학습 진단, 취약점 분석 및 맞춤형 학습 계획 제공**
*   **문제 해설, 학습 가이드 등 AI 기반 학습 지원 도구**

## 기술 스택
*   **백엔드**: FastAPI (Python)
*   **프론트엔드**: Vue.js, Vuetify
*   **데이터베이스**: MariaDB
*   **AI/NLP**: OpenAI GPT-4 API 연동, 자연어 처리(NLP)
*   **OCR**: Python 기반 PDF OCR 처리 및 문제 파싱

## 기대 효과
*   자격증 준비생의 학습 효율성 및 합격률 향상
*   AI 기술 기반의 개인화 학습 시스템 구축 노하우 습득
*   향후 다양한 교육 분야 EdTech 솔루션 확장 가능성 제공
